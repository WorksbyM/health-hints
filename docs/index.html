🌱はじめに

健康には「食事・睡眠・運動」が大切だとよく言われますが、大切だとは分かっていても実際には行動に移せていなかったことに、体調を崩して初めて気づきました。

「カラダは食べたものから作られている」その栄養は、運動によって血流が促され、全身にしっかりと届けられます。

さらに、質の良い睡眠によって自律神経が整い、心もカラダも回復していきます。

食事・睡眠・運動どれか一つだけではなく、すべてが密接に繋がっています。

それこそが、本当の「健康」なのだと、今では強く感じています。

このサイトでは、私が読んだ本の中から、実践しやすい健康のポイントを紹介していきます。

健康のポイント💡

【🥗食事編】

健康習慣 期待できる効果・理由
植物性タンパク質を選ぶ 動物性、植物性どちらにも含まれている必須アミノ酸のメチオニンの取りすぎは腎臓に負担をかけてしまう。その中でも植物性(豆類など）はほとんど吸収されないので取り入れましょう。
血糖値の急上昇を避ける 高血糖になると活性酸素が発生して血管に炎症が起こりダメージを受けた血管は厚く、硬く、脆くなってしまいます。食後に強い眠気やだるさを覚える人は糖質の取り方に気を付けましょう。
リンの種類に注意 加工食品に含まれる無機リン（インスタント麺、ファーストフード、缶詰、ハム・ベーコンなど）は吸収率が高く腎臓に負担をかけます。一方、豆類、野菜、果物などの植物性リンは吸収されにくく負担が少なめです。リンは必須ミネラルですが摂りすぎに注意し、植物性からの摂取も意識しましょう。

【🏃運動編】

健康習慣 期待できる効果・理由
スクワットやかかと上げの筋力トレーニング 下半身にはお尻や太ももなど大きな筋肉が多く、鍛えることで効率よく筋肉量を増やすことができます。特にふくらはぎは、血液を心臓に押し戻すポンプのような働きをしており、血流の改善に重要な役割を果たします。筋肉量の増加が腎臓の保護にもつながるとされています。
有酸素運動 1日20～60分を目安に週3～5回行うのが理想です。長く運動できない人は無理にせず気分転換として散歩を少し楽しむ程度でも十分です。

【🌬呼吸・生活習慣編】

健康習慣 期待できる効果・理由
寝る前の腹式呼吸 自律神経のバランスが整い、腎臓への血流が安定します。精神的にも落ち着き、寝つきが良くなり深い眠りにつながります。
水分補給は「水」が基本 スポーツドリンクは糖分が多く、緑茶やコーヒーのカフェインには利尿作用があるため、水分補給には不向きです。また「高齢になってトイレが近いから」と水分を控えると、腎機能の低下を招く恐れがあります。水分は適切に摂ることが、腎臓を守るために重要です。
サプリ・薬の摂りすぎに注意 栄養はできるだけ食品から摂ることが理想です。体調不良時もすぐ薬に頼らず、まずは食生活や体重管理など生活習慣の見直しを心がけましょう。近年では、高齢者のポリファーマシーが問題視されています。

健康のポイント💡

健康習慣 期待できる効果・理由
鼻呼吸を習慣にする 鼻は空気を浄化・加湿するフィルターで口呼吸は口が乾きやすくなって感染症の原因になったり本来の呼吸力を低下させ、さまざまな不調のもとになってしまいます。
「吐く」呼吸を意識する 私たちは1日に2万回以上呼吸をしています。呼吸は「吸う」より「吐き切る」ことが大切で深くゆっくりとした呼吸が酸素を体のすみずみに届け、自律神経を整えます。呼吸が浅く速いと、酸素の取り込み効率が下がってしまうので意識して深くゆっくりとした呼吸を心がけましょう。
舌の位置を整える 口を閉じている時の舌先は上の歯の根元あたりに当たっているもの。上あごに当て押し上げることで首や頭を支える助けになります。筋力が弱く下方に落ちてしまうと肩凝りの原因や就寝中に気道が塞がれて睡眠の質の低下に繋がってしまいます。
姿勢を見直す 長時間のデスクワークや猫背や巻き肩、スマホ首や反り腰、足組みなどは骨格を歪ませ呼吸機能にまで影響を与えてしまうので正しい姿勢を心がけましょう。
横隔膜呼吸を意識する 横隔膜には自律神経がたくさん集まっていて呼吸を通して整えることができます。吸う時には胸とお腹が同時に膨らんで吐く時には胸とお腹がへこむ自然な呼吸で体幹が安定し、代謝や血流が改善されます。

健康のポイント💡

健康習慣 期待できる効果・理由
睡眠時間を一定に保つ 平日・休日関係なく同じ時間に寝起きすることで体内リズムが整います。最低でも同じ時間に起床する。寝入りばなの90分を熟睡することで幸せホルモンのセロトニンの分泌が活発になります。睡眠の乱れが疲れのもととなり血行を悪くし基礎代謝の低下を招きます。
朝日を浴びる 体内時計のスイッチが入り14時間後くらいになにもしなくても眠気を感じ自然な眠気とスッキリした目覚めを促します。朝の光を浴びることでセロトニンを活性化し、睡眠の質が安定します。
高速腹式呼吸 息を吐いて肺の隅に溜まった古い空気を吐き出すことで肺を浄化できる。大きな筋肉が運動するので体温も上がり腹筋が鍛えられる。一瞬で空気を吐き出し吸い込むときは意識をせず自然と吸い込むを繰りかえす。
運動でリンパ循環を促す 座り仕事をしている人はリンパ液の体内循環が滞りやすくなります。リンパ液は血管の動きと筋肉の圧迫によって循環していて、特にふくらはぎを動かして血流とリンパ液の循環を促しましょう。
昼寝は10～20分 脳の疲れをとるには、机に伏せる程度の短時間の昼寝が効果的です。長時間眠ると体内リズムが乱れ、夜の睡眠に影響するため、軽く目を閉じる程度が理想です。

健康のポイント💡

健康習慣 期待できる効果・理由
生活リズムを整える 起床・食事・就寝時間を毎日同じにすることで自律神経が安定します。また、活動のオンオフをしっかり切り替えることが重要です。
深い呼吸と正しい姿勢 4秒吸って8秒吐く呼吸法で交感神経と副交感神経のバランスを整える。また、正しい姿勢（喉・胸・腹が一直線に並ぶ位置）を維持することで体幹トレーニングになります。
腸内環境を整える 栄養素がバランスよく取れると自律神経と密接な関係がある腸の働きを正常にするので足りていない栄養素はないか心がける。食品は「質」を優先させ、なるべく新鮮で食品添加物を控え、ビタミン・ミネラルも意識して摂取する。また、朝食にバナナをとることでメラトニンの分泌量を増やすことができ良質な睡眠が期待できます。
ながら運動を取り入れる 運動習慣を継続するコツは、生活の一部として自然に取り入れることです。例えば、テレビを見ながらのストレッチや、歯磨きなどの身支度をしながらの筋トレ、通勤時のウォーキング、電車でつり革を持ちながら肩甲骨を動かす運動など、日常の動作に運動を組み合わせるのも効果的です。こうした習慣によって血流が促され、酸素が全身に行きわたり自律神経のバランスが整います。
視神経を休める 蒸しタオルなどで目を覆うのが効果的ですが1分間だけ手のひらで目を覆うだけでも十分です。外界の情報の約80％は目から入ってくるので短い時間遮断するだけでもリフレッシュできます。まばたき体操・遠近体操で眼精疲労を予防しましょう。
